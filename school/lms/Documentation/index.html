What is this?

iNiLabs LMS ADAL integrate Learning management system feature into ADAL School Management System. This feature allows admins/teachers/student to create tutorial and lesson.Using this module, The teacher can provide the educational resource with the student and the student get their tutorial or lesson in tutorial module.

Features

LMS add-on, compatible with ADAL School Management System.

- Admin, teacher & student can create, delete and view
- The teacher can provide the educational resource with the student
- The student and the student get their tutorial or lesson in tutorial module
- Student can download/view the sharing resource file

Quick Start Guide

Install LMS add-on from ADAL School Admin Panel addon Module

  • Goto add-on menu under administrator menu from iNiLabs school management system admin panel
  • Upload the LMS add-on zip file.
  • It will end your login session and redirect you to the login page again
  • Now you can login again and check tutorial menu in the sidebar academic menu section.
  • Add-on successfully installed!
