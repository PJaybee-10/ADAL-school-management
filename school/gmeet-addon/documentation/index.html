What is this?

iNiLabs LMS addon integrate Gmeet Live Class feature into iNiLabs School Management System. This feature allows admins/teachers/staffs to create online live classes using Google Meet service and students can join the session from their panel accoring to class, section given schedule. This module ensures also class privacy, which means no outsiders can login into the class session without having right authority. This module is compartible with inilabs school latest version 4.7.

Features

Gmeet Live Class add-on, compatible with iNiLabs School Management System.

- Admin, teacher & users can create, delete and join Liveclass
- Students can join liveclass session from their panel as per schedule
- Secure session access directly from the panel
- Use Free Google Meet account to manage class and meetings
- Create meetings directly from school admin panel

Quick Start Guide

Install Gmeet Live Class add-on from iNiLabs School Admin Panel addon Module

  • Goto add-on menu under administrator menu from iNiLabs school management system admin panel
  • Upload the Gmeet add-on zip file.
  • It will end your login session and redirect you to the login page again
  • Now you can login again and check tutorial menu in the sidebar academic menu section.
  • Add-on successfully installed!
